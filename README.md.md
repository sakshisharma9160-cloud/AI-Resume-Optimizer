# AI-Assisted Resume & LinkedIn Optimizer

## 📌 Description
This project combines basic C++ programming with AI tools to analyze and improve resumes.

## ⚙️ Features
- Counts total words in resume
- Checks for key skills like Python and ML
- Provides basic feedback on resume length
- Uses AI tools to improve resume content

## 🔄 Workflow
Sample Resume → C++ Analysis → AI Improvement
